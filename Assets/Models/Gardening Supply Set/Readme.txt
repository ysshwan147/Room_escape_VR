#Overview
Enhance the greenery of any space with our Gardening Supplies Set. This set features a range of colourful watering cans, plant stands, and decorative pots that can complement any environment. Paired with our "Houseplant Set" series, it will bring your interior design to life.

#Contents
This package contains the following 6 models:

Watering Can (Green / Blue)
Plant Stand (3-Tier / 4-Tier)
Vase-Shaped Plant Pot (White + Geometric Pattern / Indigo + Floral Pattern)
Each model is designed with high-quality textures and details, making them ideal for real-time applications.

#Features
Detailed and realistic models perfect for enhancing indoor scenes.
Assets that complement the "Houseplant Set" and "Houseplant Set Vol. 2" for a fuller gardening experience.
Models that fit seamlessly into any environment, adding to the realism and visual appeal.

#Usage
To use these assets, import the package into your Unity project. You can then drag and drop the models into your scene as required.
